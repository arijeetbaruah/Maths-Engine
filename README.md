# 

# \# Maths Engine

# > A visual, data-driven math system for Unity that lets designers build and configure formulas using ScriptableObjects, Prefabs, and visual graphs. No code required.

# 

# \*\*Maths Engine\*\* is a Unity plugin that allows \*\*Designer\*\* to define, edit, and maintain complex mathematical formulas using \*\*visual configuration systems\*\* instead of hardcoded equations.

# 

# ----------

# \## What Problem It Solves

# 

# Traditional math in games is:

# 

# \-   Hardcoded    

# \-   Buried in scripts    

# \-   Uneditable by designers    

# \-   Fragile to changes    

# \-   Tightly coupled to systems    

# 

# Maths Engine makes formulas:

# 

# \-   Visual    

# \-   Configurable    

# \-   Data-driven    

# \-   Designer-editable    

# \-   Modular    

# \-   Reusable    

# \-   Serializable    

# \-   Testable    

# \-   Runtime-safe

# ----------

# 

# \##  Core Features

# 

# \-   Visual formula configuration    

# \-   Node-based math graphs    

# \-   ScriptableObject-based formulas    

# \-   Prefab-based configuration support    

# \-   Runtime evaluation engine    

# \-   Variable binding system    

# \-   Custom operator support    

# \-   Pluggable math nodes    

# \-   Async evaluation support    

# \-   Validation \& error detection    

# \-   Serialization support    

# \-   Designer-safe tooling    

# \-   Engine-agnostic core

# 

# ----------

# 

# \## Philosophy

# 

# \*\*Code defines structure. Data defines behavior.\*\*

# 

# This plugin follows a simple rule:

# 

# > Programmers build the system. Designers build the logic.

# 

# Formulas are not code.  

# They are \*\*assets\*\*.

# 

# This allows:

# 

# \-   Live balancing    

# \-   Rapid iteration    

# \-   No recompiles    

# \-   No code churn    

# \-   Safer tuning    

# \-   Easier debugging    

# \-   Better collaboration

# 

# \## Quick Start

# 

# 1\.  Create a \*\*Formula Asset\*\*    

# 2\.  Add math nodes (Add, Multiply, Clamp, Curve, etc)    

# 3\.  Bind variables (ATK, DEF, Level, etc)    

# 4\.  Save as ScriptableObject    

# 5\.  Evaluate at runtime: 

# 

# float  result  =  formula.Evaluate(context);
