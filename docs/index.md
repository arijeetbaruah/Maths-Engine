\# Maths Engine



Maths Engine is a visual, data-driven math system for Unity that allows designers to configure formulas using assets instead of code.



\## What It Does

\- Visual formula graphs

\- ScriptableObject-based configuration

\- Runtime evaluation engine

\- Variable binding system

\- Designer-friendly workflows
