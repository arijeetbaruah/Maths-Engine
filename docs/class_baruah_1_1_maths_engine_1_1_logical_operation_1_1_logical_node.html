Detailed Description

A math node that evaluates a chain of comparison expressions and returns one of two values.

Each ComparisonEntry evaluates a comparison between two math nodes. Entries can be chained together using logical operators such as AND, OR, or XOR.

After the comparison chain is evaluated, the node behaves like a ternary expression:

condition ? True : False

If the final result of the logical evaluation is true, the True node is evaluated and returned. Otherwise the False node is evaluated.

override string Baruah.MathsEngine.LogicalOperation.LogicalNode.ToEquation ( )

Builds a parenthesized ternary equation string representing this logical node.

The generated equation follows this format:

(A > B AND C < D ? TrueValue : FalseValue)

Null nodes are represented as 0 and missing operators as ?.
